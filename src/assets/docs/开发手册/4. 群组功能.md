# 1、群组功能
## 1.1、创建群组
SIM提供创建私有群的功能，人数上限2000人。通过接口[`SIM.createGroup(config, onSuccess, onFail)`]()，如下：

```
window.SIM.createGroup({
	type: 0, // 群组类型
	name: '测试群', // 群组名字,
	members: ['10000', '10001', '10002'], // 群组成员列表
}, (e) => {
	console.log('sendMsg end', e);
}, (err) => {
	console.log('sendMsg err', err);
});
```
